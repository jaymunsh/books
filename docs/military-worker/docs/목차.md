# 눈 떠보니 9급 군수직 군무원 목차

## 책의 방향

이 책은 2026년 기준 9급 군무원 군수직 공채를 준비하는 일반인을 위한 합격 매뉴얼이다.
7급, 경채, 지역인재, 다른 직렬은 부록 수준으로만 다루고, 본문은 9급 군수직에 집중한다.

기출문제와 문제풀이 파트는 책 본문과 별도 HTML로 확장한다.
본문은 이해와 전략, 문제풀이 HTML은 실제 풀이와 오답 관리에 집중한다.

---

## 프롤로그. 이 책의 목표: 9급 군수직 합격

1. 이 책은 군무원 입문서가 아니라 합격 매뉴얼이다
2. 9급 군수직만 다루는 이유
3. 이 책을 보는 순서: 자격 확보 → 전략 선택 → 과목 공부 → 실전 → 면접
4. 독자가 이 책으로 해야 할 최종 결과물
