**거버넌스·표준화는 없습니다.** 가장 적은 노력으로 가장 큰 효과만 챙깁니다.

이 파일 하나 + 커맨드 두세 개면 충분합니다. 팀이 커지면 그때 늘리세요.

## 프로젝트
Next.js + Supabase 로 만든 개인 가계부 앱.

## 규칙
- 금액은 원화 정수. 소수점 금지.
- 컴포넌트는 `src/components/`, 페이지는 `src/app/`.
- 새 의존성을 추가하기 전에 먼저 물어본다. (번들 크기가 중요)
- 커밋 전에 `npm run build`가 통과해야 한다.

## 명령
- 개발: `npm run dev`
- 빌드(=검증): `npm run build`
- 타입체크: `npm run typecheck`

## 하지 말 것
- Supabase 프로덕션 테이블 직접 수정 금지. 마이그레이션으로.
- `.env.local` 읽기 금지.
