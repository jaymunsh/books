# 브랜드 보이스 가이드

대량 생성에서 가장 먼저 무너지는 것이 **톤의 일관성**입니다.
작성자가 10명이든 에이전트가 100건을 만들든, 목소리는 하나여야 합니다.

## 우리 목소리

- **명료함 > 화려함.** 어려운 말을 쉽게 풀어 쓰는 것이 우리의 강점입니다.
- **독자를 낮춰보지 않습니다.** "쉽게 말하면"보다 그냥 쉽게 말합니다.
- **1인칭 복수(우리)를 씁니다.** "저희"는 과도하게 겸양적이라 쓰지 않습니다.
- 문장은 짧게. 한 문장에 한 가지만.

## 쓰지 않는 표현

| 쓰지 않음 | 대신 |
|-----------|------|
| "혁신적인", "게임체인저" | 무엇이 어떻게 달라지는지 구체적으로 |
| "~라고 할 수 있습니다" | "~입니다" (단정) |
| "많은 분들이" | 실제 수치 또는 삭제 |
| 느낌표 남발 | 한 글에 최대 1개 |
| 이모지 남발 | 제목엔 쓰지 않음, 본문 최대 2개 |

## 구조 규칙

1. 첫 문단에서 **독자가 얻을 것**을 말합니다. 배경 설명으로 시작하지 않습니다.
2. 소제목은 내용을 요약합니다. "들어가며" 같은 빈 제목을 쓰지 않습니다.
3. 마지막에 다음 행동을 제시합니다.

## 검수 시 확인

- [ ] 첫 문단이 효용으로 시작하는가
- [ ] 금지 표현이 없는가
- [ ] 근거 없는 수치·주장이 없는가
- [ ] 문장이 불필요하게 길지 않은가

> ⚠️ 이 스킬은 **생성**용입니다. 검수는 별도 에이전트가 수행합니다.
> 같은 에이전트가 쓰고 검수하면 자기 편향이 생깁니다(본편 6.3 참조).
