---

작업을 "완료"로 보고하기 전에 아래를 **실제로 실행**하고 결과를 붙여라.
통과하지 못한 항목이 하나라도 있으면 완료라고 말하지 마라.

1. 테스트:    !npm test
2. 타입체크:  !npm run typecheck
3. 린트:      !npm run lint

## 규칙
- 각 명령의 **실제 출력**을 요약해 보고한다("아마 통과할 것" 금지).
- 실패가 있으면 원인을 고친 뒤 이 점검을 처음부터 다시 반복한다.
- 세 항목이 모두 통과한 뒤에야 완료를 보고하고, 통과 개수를 함께 적는다.
  (예: "테스트 27개 통과, 타입 에러 0, 린트 경고 0")
