# MarketFlow 웹앱

전자상거래 플랫폼의 프론트엔드(React) + 백엔드 API 저장소입니다.
이 파일은 세션마다 자동으로 읽힙니다. 짧고 구체적으로 유지하세요.

## 프로젝트 규칙
- 통화는 항상 원화(KRW) **정수**로 다룬다. 소수점·부동소수 금지.
- 상품 가격은 `price_krw` 필드를 쓴다. `price`, `amount` 금지.
- 모든 API 응답은 `snake_case`. 프론트 변환은 어댑터 계층(`src/adapters/`)에서만.
- 날짜는 ISO 8601 + KST(+09:00)로 저장한다.
- 함수는 40줄을 넘기지 않는다. 넘으면 분리한다.

## 자주 쓰는 명령
- 개발 서버: `npm run dev`
- 테스트: `npm test` — 변경한 코드는 반드시 테스트를 돌린 뒤 완료를 보고한다.
- 타입체크: `npm run typecheck`
- 린트/포맷: `npm run lint`

## 커밋 규칙
- 메시지는 `feat:`, `fix:`, `chore:`, `docs:` 프리픽스로 시작한다.
- 결제(`src/payments/`) 관련 변경은 사람 리뷰 없이 커밋하지 않는다.

## 하지 말 것
- 프로덕션 DB에 직접 쿼리 금지. 스키마 변경은 반드시 마이그레이션으로.
- 비밀키·토큰은 `.env`에서만 읽는다. 절대 하드코딩하지 않는다.
- `legacy/` 폴더는 리팩토링 대상이 아니다. 그 폴더의 규칙을 따른다.
