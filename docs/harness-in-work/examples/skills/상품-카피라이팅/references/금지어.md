# 카피 금지어·주의어 목록

카피 작성 시 아래 표현은 피하거나 반드시 [심의 확인] 처리한다.
표시광고법·전자상거래법 위반 위험이 있는 표현 위주로 정리한다.

## 절대 금지 (최상급·절대성)
- 최고, 최상, 최고급, 업계 1위(근거 없을 때)
- 유일무이, 유일한, 오직
- 100%, 완벽, 무조건, 절대
- 세계 최초·국내 최초(공신력 있는 근거 없을 때)

## 주의 (근거 필수)
- "인기", "베스트셀러", "품절 임박" — 실제 데이터 근거가 있을 때만
- "친환경", "천연", "무해" — 인증·시험 결과가 있을 때만
- 할인·특가 — 원가·정가 근거가 명확할 때만

## 카테고리별 특수 주의
- 건강/식품: 질병 예방·치료·효능 표현 금지(의약품 오인 위험)
- 화장품: "재생", "개선" 등 기능성 표현은 인증 범위 내에서만
- 아동/유아: 안전 관련 과장 표현 특히 주의
