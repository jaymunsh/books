# 상품 카피라이팅 규칙

## 원칙
- 첫 문장에 핵심 효용을 넣는다. 미사여구로 시작하지 않는다.
- 구체 수치로 말한다: "빠른 배송" 대신 "평균 1.8일 배송".
- 사용자가 얻는 것을 말한다. 스펙 나열이 아니라 효용 번역.
- 한 문단은 한 가지 메시지만 담는다.

## 금지
- 과장·최상급 표현(최고, 유일무이, 100% 등) — 광고 심의 위반 위험.
  전체 목록: references/금지어.md
- 경쟁사 비방·비교 표현.
- 근거 없는 효능·효과 주장(특히 건강·식품 카테고리).

## 출력 형식
1. 한 줄 후킹 문구(30자 이내)
2. 핵심 효용 3개(불릿)
3. 상세 설명 문단
4. 심의 리스크가 있는 표현을 썼다면 맨 아래 `[심의 확인]`으로 표시
